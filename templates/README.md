# Card Icon Templates

This folder should contain card icon templates for template matching.

## How to populate:

1. Fetch card icons from Clash Royale API
2. Save each card icon as: `{cardName}.png` or `{cardName}.jpg`
3. Example: `hog_rider.png`, `fireball.png`, etc.

## Template Matching:

The system will use these templates to detect cards in video frames.

## Auto-population:

The system can automatically download and store card icons from the Clash Royale API when needed.
